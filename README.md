## 🎯 一句话说明

收到一条新闻，直接出稿：**这事到底怎么了、会怎么发展、对谁有影响、你应该怎么做。**

不是复述新闻，是给观点。不是情绪发泄，是决策参考。

---

## ✨ 核心能力

- ✅ **原新闻+来源**：保留原始信息，出处可查
- ✅ **深度推演**：这事1-3个月会怎样？半年到1年呢？
- ✅ **决策建议**：分投资者/从业者/普通用户，给出具体行动
- ✅ **金句收尾**：一句话升华，让人看完想转发

---

## 🚀 安装使用

### OpenClaw CLI 安装
```bash
# 安装 skill
clawhub install ruiping-skill

# 查看帮助
cat SKILL.md
```

### 直接使用
```
激活后，直接丢新闻给我 → 直接出稿
```

**最快方式**：直接丢新闻 → 直接出稿

**搜索追踪**：说「最近AI领域有什么值得评的大事」→ 搜完筛完出稿

**懒人模式**：说「最近有什么值得评的大事？」→ 帮你找热点并筛选

## 📝 标准输出格式

```
📌 【锐评】[事件名称]

📰 原新闻
[原文核心内容摘要]

📎 来源：[新闻出处]

一句话结论：[最核心的判断，20字以内]

▌这事到底怎么了
[拆穿表象，说清本质逻辑]

▌会怎么发展
[短期1-3月+中期半年到1年的推演]

▌对谁有影响，影响多大
- [影响对象1]：[具体机制、量级]
- [影响对象2]：[同上]

▌你应该怎么做
- 如果你是投资者：[具体建议]
- 如果你是从业者：[具体建议]
- 如果你是普通用户：[具体建议]

▌金句
[一句话升华，让人看完想转发]
```
