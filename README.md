# 锐评 🔥

> **你每天看50条新闻，有几条真正帮你做判断了？**

[![虾评](https://img.shields.io/badge/虾评-安装-blue)](https://xiaping.coze.site/skill/fbd59253-f9ed-4628-975b-b0671b768a1f?ref=8fa30089-2274-441a-a0c3-b2bd9aab0f8c) [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

别人追热点，你追观点。

---

## 为什么需要锐评？

信息爆炸的时代，你缺的不是新闻，是判断力。

每条热点都在告诉你"发生了什么"，但没人告诉你"意味着什么"、"对你有什么影响"、"你该怎么做"。

锐评不负责给你更多信息，负责给你**更深的判断**。

> 看完50条新闻你还是你，看完1条锐评你多想了一层。

---

## 怎么用

```
"锐评一下特斯拉降价"
"帮我追踪AI大模型的热点，筛值得评的"
"最近有什么值得评的大事？"
```

三种姿势：你给事件我评 / 你给方向我找 / 懒人模式我全包。

---

## 三步走

```
追踪热点 → 筛选值得评的 → 给出犀利观点
```

### 1. 追踪去哪找？

你只管说"关注新能源车最近动态"或者"帮我盯着AI大模型竞争"，剩下的我来：

- **全网热点**：微博热搜、知乎热榜、抖音热榜、百度热搜
- **行业垂直**：36氪、虎嗅、钛媒体、晚点LatePost
- **财经动态**：雪球、同花顺、东方财富
- **国际大事**：BBC中文、纽约时报中文

### 2. 什么值得评？三个筛选标准

- 🎯 **影响判断型**：直接影响你决策的事——政策变化、行业洗牌、重要人事变动
- 🤔 **反直觉型**：看起来对但仔细想不对，或者所有人都想错了的事
- 🔮 **趋势预判型**：现在是小苗头，但3年后会是大事的信号

### 3. 怎么评出价值？

每条锐评必须回答三个问题：
- 这事到底**意味着什么**？（不只是表面发生了什么）
- 对谁有影响？**影响多大**？
- 普通人**该怎么做**？

## 输出示例

**你说的**："特斯拉又降价了，这次降了3万"

**我出的**：

> 📌 **【锐评】特斯拉又降价：价格战打到最后，比的是谁亏得起**
> 
> **一句话结论**：这不是在抢市场，这是在逼死对手。
> 
> ▌**为什么重要**
> - 这次降价直接击穿了20万心理关口，电动车和燃油车的最后一道价格墙没了
> - 特斯拉宁可牺牲利润也要保市占率，说明订单压力比财报看到的更严峻
> 
> ▌**对谁有影响**
> - 蔚来/小鹏/理想：被迫跟进，利润进一步压缩
> - 比亚迪：有成本优势可以硬扛，但也被拖入消耗战
> - 等等党：赢了，但现在入场时机也到了
> 
> ▌**该怎么做**
> - 如果你是等等党：今年上半年是最佳购车窗口
> - 如果你在观望：别等Model 2了，先买现款
> - 如果你是车企员工：准备好简历，这次洗牌会很残酷
> 
> ▌**一句话锐评**
> 特斯拉不是在卖车，是在卖"我还能撑多久"的信仰。
