# 锐评 🔥

> **你每天看50条新闻，有几条真正帮你做判断了？**

别人追热点，你追观点。

## 怎么用

- "锐评一下特斯拉降价"
- "帮我追踪AI大模型的热点"
- "最近有什么值得评的大事？"

## 三步走

1. **追踪** → 告诉我关注什么领域，我来找热点
2. **筛选** → 影响判断的/反直觉的/趋势预判的
3. **评论** → 不复述新闻，给观点、给判断、给行动建议

## 输出示例

📌 特斯拉又降价：这不是在抢市场，这是在逼死对手。

- 为什么重要：击穿20万心理关口，电动车和燃油车最后一道价格墙没了
- 对谁有影响：蔚来/小鹏被迫跟进，等等党赢了
- 该怎么做：今年上半年最佳购车窗口，价格战子弹快打光了
- 一句话锐评：特斯拉不是在卖车，是在卖"我还能撑多久"的信仰

> 看完这条，你比99%的人多想了一层。
