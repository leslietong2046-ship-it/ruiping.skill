# 锐评：你每天看50条新闻，有几条真正帮你做判断了？

> **别人追热点，你追观点。**

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://img.shields.io/badge/License-MIT-blue.svg)

---

## 🎯 一句话说明

收到一条新闻，直接出稿：**这事到底怎么了、会怎么发展、对谁有影响、你应该怎么做。**

不是复述新闻，是给观点。不是情绪发泄，是决策参考。

---

## 🆕 v3.0 重大升级：从热点搬运到决策武器

**升级前**：给个框架让你自己填 → **升级后**：直接出完整锐评，可直接使用

- ✅ **原新闻+来源**：保留原始信息，出处可查
- ✅ **深度推演**：这事1-3个月会怎样？半年到1年呢？
- ✅ **决策建议**：分投资者/从业者/普通用户，给出具体行动
- ✅ **金句收尾**：一句话升华，让人看完想转发

## 🔑 核心方法：三步走

```
筛选 → 分析 → 评论
```

**什么值得评？三个筛选标准：**
- **影响判断型**：直接影响你决策的事——政策变化、行业洗牌、重要人事变动
- **反直觉型**：看起来对但仔细想不对的事
- **趋势预判型**：现在是小苗头，但3年后会是大事的信号

---

## 💡 关键原则

- **直接输出**：收到素材就出稿，不问"需要我评吗"
- **不废话**：不要"让我来分析一下"这种废话开头
- **有态度**：结论要犀利，不能模棱两可
- **有推演**：不能只评当前，必须推演走向
- **有决策价值**：读完好的人应该知道自己下一步该做什么
