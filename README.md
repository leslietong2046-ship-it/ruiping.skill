# 锐评：你每天看5条新闻，有几条真正帮你做判断了？

> **别人追热点，你追观点。**

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://img.shields.io/badge/License-MIT-blue.svg)

---

## 一句话说明

收到一条新闻，直接出稿：**这事到底怎么了、会怎么发展、对谁有影响、你应该怎么做。**

不是复述新闻，是给观点。不是情绪发泄，是决策参考。

---

## 🆕 v3. 重大升级：从热点搬运到决策武器

**升级前**：给个框架让你自己填 → **升级后**：直接出完整锐评，可直接使用

- **原新闻+来源**：保留原始信息，出处可查
- **深度推演**：这事1-3个月会怎样？半年到1年呢？
- **决策建议**：分投资者/从业者/普通用户，给出具体行动
- **金句收尾**：一句话升
