# 锐评：你每天看50条新闻，有几条真正帮你做判断了？

别人追热点，你追观点。

---

## 什么是锐评

不是新闻搬运，是观点输出。不是情绪发泄，是决策参考。

收到一条新闻，直接出稿：这事意味着什么、会怎么发展、对谁有影响、你应该怎么做。

## v3.0 更新内容

- **📰 原新闻 + 📎 来源**：保留原始信息，出处可查
- **🔍 深度推演**：这事1-3个月会怎样？半年到1年呢？
- **💡 决策建议**：分投资者/从业者/普通用户，给出具体行动

## 使用方式

**最快**：直接丢新闻给我 → 直接出稿

**搜索追踪**：说「最近XX领域有什么值得评的大事」→ 我搜完筛完出稿
